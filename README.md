# 音の波形とスピーカーの仕組み学習ビジュアライザー

中学校技術科(情報の技術)の授業向けに作成した、音の波形をリアルタイムに可視化するWebアプリです。
マイクの声・発振器のテスト音・音楽ファイルの3つの音源を切り替えながら、電気信号(波形)がスピーカーを動かして音になる仕組みを視覚的に学べます。

## 🔊 できること

- **3つの音源から選べる**
  - マイク入力(自分の声やスピーカーの音を拾う)
  - 発振器(サイン波・三角波・四角波・ノコギリ波、周波数プリセット付き)
  - 音楽ファイルの読み込み再生
- **3つの観察モード**
  - オシロスコープ(波形そのものを表示)
  - 周波数別表示(低音・中音・高音の強さを表示)
  - レトロメーター(昭和オーディオ風の強弱インジケータ)
- 波の色や感度(拡大率)を自由に変更可能
- 音の大きさ・音の高さ(音名)・一番強い周波数を生徒向けにわかりやすく数値表示
- RMS・波高値・デシベル値・サンプリング周波数などの詳細データも表示(先生・発展学習用に折りたたみ表示)

## 📖 使い方

1. 下記のURLをブラウザで開く(スマートフォン・タブレット・PCいずれも可)
   ```https://playa2021git.github.io/audio-waveform-display/```
2. 「マイク接続」を押し、ブラウザのマイク使用許可を「許可」する
3. 声を出したり、近くでスピーカーを鳴らしたりして波形の変化を観察する
4. 表示モードや感度、色を変えて見比べてみる

## ⚠️ 注意事項

- マイク機能は **HTTPS環境が必須**です(GitHub Pagesは自動的にHTTPS対応のため問題なく動作します)。
- マイクの使用許可を求めるダイアログが出ない場合は、ブラウザのアドレスバー付近のマイクアイコンから許可設定を確認してください。
- 対応ブラウザ:最新版の Google Chrome / Microsoft Edge / Safari を推奨します。

## 🛠️ 技術構成

- HTML / CSS / JavaScript(フレームワーク不使用)
- [Web Audio API](https://developer.mozilla.org/ja/docs/Web/API/Web_Audio_API)(音声解析・発振器・波形描画)
- [Tailwind CSS](https://tailwindcss.com/)(CDN版)
- [Font Awesome](https://fontawesome.com/)(アイコン)
